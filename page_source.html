オンラインチケット 購入日画面
  • カレンダーにて来館日を選択してください。
  • 来館日を選択すると次の画面で時間を指定できます。
  • 開館日の内、定員に達した日は選択できません。
  • カレンダーで日時を選択しても、予約が集中した場合などは次画面にてチケットを確保できない場合がございます。
  • 予約完了後、「お申し込みが完了しました」メールをお送りいたします。「@eventos.tokyo」からのメールを受信できるようあらかじめ設定をお願いします。

インフォメーション

開館日時・休館日
詳細は 豊島美術館ページ をご確認ください。
鑑賞料（税込）
通常入館 オンライン購入 1,800円、窓口購入 2,000円
※15歳以下無料。

「朝の特別鑑賞プログラム」付入館チケット 2,300円
年間パスポート所有者用「朝の特別鑑賞プログラム付」入館チケット 500円
※15歳以下、島民無料。
※朝の特別鑑賞プログラムは不定期開催のため、詳細は豊島美術館ページをご確認ください。
チケット種別
  • 本券は、30分毎の日時指定の入館予約チケットです。（繁忙期は、15分毎の日時指定の入館予約チケットです。）朝の特別鑑賞プログラムの場合限定的な日時指定の入館予約チケットです。
  • チケット販売期間は2カ月前の第二金曜10:00～予約時間までです。
  • 入館人数は、各時間枠で制限がございます。
  • 指定日の予約時間のみ有効です。

  • 朝の特別鑑賞プログラムの集合時間は8:45～9:00となります。
  • 朝の特別鑑賞プログラム終了後、10:00から通常開館いたします。

スマートフォン画面表示
  • 予約後、 マイページ より「マイチケット」からQRコードを表示することができます。または、「お申し込みが完了しました」メールから表示することも可能です。
  • チケットは1人につき1つのQRコードが表示されます。
  • ご利用当日は、スマートフォンの充電が十分な状態でご来場ください。

ご自身でプリントアウト
  • 予約後、 マイページ 内にある「マイチケット」からQRコードを印刷してください。または、「お申し込みが完了しました」メールから印刷してください。
  • チケットは1人につき1つのQRコードが表示されます。
  • チケットはQRコードが鮮明に印字されていることをご確認ください。
  • チケットの印刷はモノクロ印刷でも問題ありません。

オンラインでの購入
  • クレジットカードのみご利用可能です。
    VISA / MASTER CARD / JCB / AMERICAN EXPRESS / DINERS / DISCOVER

窓口での購入
  • ご予約なしで来館された場合は、ご希望の時間枠に空きがあれば当日窓口（チケットセンター）でご購入いただけます。
    ただし、完売の場合はご購入いただけませんので、オンラインチケットの事前購入をお勧めします。

    現地でのお支払いは現金・クレジットカード（VISA / MASTER CARD / JCB / AMERICAN EXPRESS / DINERS / DISCOVER）・ 電子決済（Alipay / WeChat Pay / PayPay / au PAY / メルペイ / d払い / LINE pay）・ 電子マネー（交通系IC / QUICPay / nanaco / WAON / iD）です。
    窓口でのご購入の場合、オンライン購入とは価格が異なります。詳しくは上記の「鑑賞料金」をご確認ください。
    なお、窓口でご購入いただいたチケットの払い戻し、および再発行はご対応できかねますのでご了承ください。

日時変更
  • チケットの日時変更はできません。一度キャンセルいただき、再度予約してください。

キャンセル
  • キャンセルの受付は、ご予約時間の30分前までです。
    例：ご予約時間が11:00の場合は10:30までです。
  • チケットの日時変更はできません。一度キャンセルいただき、再度予約してください。
  • 受付期間後のキャンセルはお受けすることができません。
  • 「@eventos.tokyo」からのメールを受信できるようあらかじめ設定をお願いいたします。
  • キャンセル手数料はかかりません。
  • キャンセルの取消はできません。
  • クレジットカードで購入された場合、代金はクレジットカードを通じて返金されます。

  • 予約でご入力いただいた内容にお間違いがないようご注意ください。予約後の変更はできません。
  • 予約中にブラウザの「戻る」ボタンを操作すると、予約が正常に行われない場合がございます。
